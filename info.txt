SEMI		;
STATIC		static
COMMA		,
RETURN		return
PLUS		\+
MINUS		-
TIMES		\*
OVER		/
RAISE		\*\*
[ignored]	[\f\t\r\n]+

=== SYNTAX ===
expr: expr PLUS/MINUS/TIMES/OVER/RAISE/COMMA expr
expr: IDENT / NUMBER
